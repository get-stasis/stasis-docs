# Stasis CI

A distributed CI/CD runner system built with Rust, designed to execute CI jobs in isolated Docker containers. Integrates with Stasis Git server.

## Features

<Cards>
  <Card title="HTTP API" description="RESTful API for job submission and management" />
  <Card title="Docker Isolation" description="Each job runs in an isolated Docker container" />
  <Card title="Git Integration" description="Secure repository cloning with authentication" />
  <Card title="Real-time Logging" description="Stream logs to Git Server in real-time" />
  <Card title="Artifact Management" description="Collect, compress, and store build artifacts" />
  <Card title="Job Replay/Retry" description="Re-run jobs with same or modified configuration" />
</Cards>

## Quick Start

```bash
# Navigate to CI runner directory
cd stasis-ci

# Setup environment
cp .env.example .env

# Start development environment
make dev-up

# View logs
make dev-logs
```
